What this project does

Fast clarity when a message feels off

Simple tools that help you understand tone, spot patterns, and respond with more confidence.

AI Message Scanner

Checks tone, insults, and manipulation cues so you can see what the message is really doing.

Smart Reply Generator

Suggests calmer, safer responses when you want to set a boundary without escalating the situation.

Detect Red Flags

Highlights patterns like exclusion, threats, repeated mocking, and guilt-driven language.

Confidence Boost

Turns confusing messages into clearer guidance so you can act with more confidence and less self-doubt.

Project pillars

The four core parts of the platform

These are the main building blocks that shape how the project supports awareness, safety, and action.

Education

Clear and detailed explanations of bullying and cyberbullying to build awareness and understanding.

Smart Detection

A text analyzer that identifies harmful language and highlights potential bullying behavior.

Safe Vault

A secure space where detected messages are stored as evidence for future reference.

Report System

A structured reporting system that allows users to safely report bullying incidents.
